## 🚀 What This Project Does

This workflow automates the entire outbound pipeline:

**Seed Profiles / Channels → Related Prospect Discovery → Data Enrichment → Personalized Outreach → Reply & Booking Ready**

### Core Capabilities
- Scrapes **YouTube channels / LinkedIn profiles** as seed inputs  
- Finds **25+ highly related prospects per run**
- Scales to **100+ qualified leads automatically**
- Enriches leads with:
  - Email (when available)
  - Channel stats (subs, views)
  - Descriptions & metadata
- Uses **AI to write hyper‑personalized cold emails**
- Sends emails via **SMTP (Brevo / Gmail / Outlook)**
- Stores everything in **Google Sheets**
- Protected with **license / usage limits**
- Ready for **reply tracking & meeting booking**

## 🧩 Tech Stack

- **n8n** – Automation & orchestration

- **OpenAI / LLMs** – Personalization
